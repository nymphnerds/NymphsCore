Workflow

A direct Nano Banana workflow for Blender.

FlashAddon is a direct Blender workflow: prompt an image, optionally guide it with another image, plan parts from a master character reference, and extract only the parts you want.

Step 01

Features

Four core features in one Blender addon.

This first `flash.html` page is meant to introduce the addon clearly and quickly. It is a standalone product page scaffold, ready for real screenshots, install steps, and release notes.

✦
Prompt To Image
Run Nano Banana image requests directly from Blender with model, aspect, and size controls in the same panel.
◎
Guide-Image Mode
Pick an existing image and use it as a visual anchor when you need edits, not fully fresh generations.
▦
Part Planning
Use Gemini vision to propose extractable parts from a master character image before spending on edits.
⬢
Part Extraction
Select only the parts you want, keep symmetry when needed, and save isolated outputs with metadata.
